# Phase 0 - Foundations

## Purpose
Build networking, Linux, HTTP, note-taking, and first safe-lab workflows.

## Weeks in this phase
- Week 01: Environment setup, networking baseline, first lab notes
- Week 02: Subnetting, host communication, Linux users, and first repeatable scanning workflow
- Week 03: Authentication, common vulnerabilities, and first exploit attempts in guided labs
- Week 04: Foundation phase capstone and first machine-style writeup

## Phase milestones
- Start the phase with explicit weekly notes, deliverables, and quick-reference updates.
- End the phase with a checkpoint note and at least one report-quality artifact.

## What success looks like by the end of the phase
- Networking, Linux, HTTP, and Burp basics feel operational instead of abstract.
- The vault is usable as an execution system.
- At least one hands-on lab note and one machine-style writeup exist.
