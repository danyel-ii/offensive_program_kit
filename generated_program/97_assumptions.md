# Assumptions

- The candidate uses Kali, Ubuntu, WSL, or an equivalent Linux environment that supports bash, nmap, Burp Suite, and standard training-lab tooling.
- Stable platform categories are preferred over brittle retired room or box names when exact names are likely to drift.
- Lab IP addresses and target names are supplied by the training platforms or the local lab at execution time.
- The user will manually update the dashboard's current week pointer as they progress.
- OffSec, THM, HTB, and PortSwigger access already exist or will be created during Week 1 setup.
- Optional plugin snippets are avoided; the vault is plain Markdown first.
