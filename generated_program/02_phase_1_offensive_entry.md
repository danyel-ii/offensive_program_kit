# Phase 1 - Offensive Entry

## Purpose
Build repeatable enumeration, first exploit attempts, and first real machine writeups.

## Weeks in this phase
- Week 05: Enumeration habit with TCM PEH and structured beginner offensive labs
- Week 06: Consolidation week for enumeration, Burp basics, and first repeatable target workflow
- Week 07: Web exploitation basics with SQLi, XSS, auth logic, and directory discovery
- Week 08: Phase 1 capstone with a full beginner attack chain and cleaner reporting

## Phase milestones
- Start the phase with explicit weekly notes, deliverables, and quick-reference updates.
- End the phase with a checkpoint note and at least one report-quality artifact.

## What success looks like by the end of the phase
- Enumeration has become a routine rather than an improvisation.
- Web basics and first exploit attempts are documented clearly.
- Machine writeups and exploit notes exist every week.
