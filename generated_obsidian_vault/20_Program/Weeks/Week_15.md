# Week 15 - PEN-200 onboarding, Proving Grounds rhythm, and OSCP-style note discipline

**Phase:** Phase 3 - OSCP Track
**Weekly totals:** 1080 active minutes / 1200 elapsed minutes
**Review note:** [[30_Reviews/Weekly/Week_15_Review]]

## Weekly focus
- start the OSCP-track phase with exam-oriented note and target habits
- shift to Proving Grounds or current TJ Null-style target progression
- keep reports and quick references aligned to real targets

## Expected weekly outcomes
- candidate has a clearer OSCP-style note structure
- one PG or current TJ Null-aligned easy target is worked with report intent
- PEN-200 becomes a regular structured anchor

## Saturday - 240 active minutes / 270 elapsed minutes
### Hour 1 - 60m active
**Resource:** Second easy PG or TJ Null-aligned target
**Task:** Do a fresh first-pass enumeration and keep notes compact from the start.
**Tools:** nmap, ffuf or gobuster, browser
**Output:** Second target note
**Success criteria:** You get to ranked hypotheses efficiently and cleanly.

### Break - 10m
### Hour 2 - 60m active
**Resource:** Exploit work
**Task:** Pursue the best access path and record only useful evidence and pivots.
**Tools:** bash, Burp Suite, browser
**Output:** Exploit attempt note
**Success criteria:** The note avoids clutter and preserves decision-making clarity.

### Break - 10m
### Hour 3 - 60m active
**Resource:** Post-access and privesc
**Task:** If access is gained, work the relevant checklist and capture the attack chain in report-ready terms.
**Tools:** linpeas or winpeas, bash
**Output:** Privesc note
**Success criteria:** The note is selective and report-oriented.

### Break - 10m
### Hour 4 - 60m active
**Resource:** Learning log
**Task:** Write down what changed when you began writing notes as if another assessor would read them.
**Tools:** Obsidian
**Output:** Learning log: `Week 15 - Note Discipline`
**Success criteria:** The note identifies concrete reporting improvements.

### Outputs
- Second target note
- Exploit attempt note
- Privesc note
- Learning log: `Week 15 - Note Discipline`

### Session success criteria
- You get to ranked hypotheses efficiently and cleanly.
- The note avoids clutter and preserves decision-making clarity.
- The note is selective and report-oriented.
- The note identifies concrete reporting improvements.
