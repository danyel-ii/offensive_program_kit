# Missed Session Protocol

## Purpose
Recover without destroying the next week, overloading short days, or sacrificing the practical core.

## Priority order when a session is missed
1. Preserve one practical lab or target session for the week.
2. Preserve one output artifact: exploit note, machine writeup, or report fragment.
3. Preserve the weekly review.
4. Drop or defer passive review before cutting hands-on work.

## If a 2-hour day is missed
- Move one concept-summary block into the next 2-hour day only if that next day is not already practical-heavy.
- Do not move both hours blindly.
- Keep the week's main target session intact.
- If needed, compress by removing repeated review or trimming note-polish time.

## If a 4-hour day is missed
- Do not cram four active hours into a 2-hour day.
- Salvage one key practical objective into the next 4-hour day, such as enumeration plus one exploit-validation block.
- Defer secondary work like note cleanup, extra warm-up labs, or second-target volume into the next consolidation window.

## If a full week goes off track
- Preserve these three items only:
  1. one practical target or lab
  2. one artifact
  3. one weekly review
- Skip low-value passive review and optional second-target work.
- Resume the following week on schedule; do not roll the entire missed week forward.

## What to drop first
- Duplicate review blocks
- Extra screenshot cleanup
- Second-target volume
- Nice-to-have note polishing

## What not to drop first
- The week's main sanctioned practical session
- The exploit note or machine writeup
- The weekly review
- The phase checkpoint or mock debrief when scheduled

## Friday rule
- Friday stays rest.
- Do not use Friday to cram missed work.

## Recovery examples
- Missed Monday theory block: move the concept summary into Thursday or next Tuesday, but keep Saturday's practical session unchanged.
- Missed Saturday machine block: preserve one key target block on Sunday and defer writeup polishing or second-target work.
- Missed mock continuation: keep one full timed target with same-day reporting and the debrief instead of inflating scope.
