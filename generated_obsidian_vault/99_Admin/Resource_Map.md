## Link usage note
- These are stable landing pages or course pages.
- Internal module, room, and lab URLs often require login and can change; use the platform landing page if a deep link no longer matches the current catalog.

## Phase 0 - Foundations
- Jason Dion Network+ starts immediately in Weeks 1-4 for networking vocabulary, protocols, services, and subnetting.
- Jason Dion Security+ starts immediately in Weeks 1-4 for attack surface, auth, secure configuration, and basic security framing.
- Linux Basics for Hackers runs in Weeks 1-4 for Linux comfort and shell muscle memory.
- PortSwigger Web Security Academy starts in Week 1 and repeats weekly for HTTP, auth, SQLi, XSS, and request logic.
- TryHackMe foundational rooms or the closest stable current equivalents start in Week 1 for safe hands-on practice.

## Phase 1 - Offensive Entry
- TCM Practical Ethical Hacking becomes the primary guided offensive resource in Weeks 5-8.
- TryHackMe beginner offensive, Nmap, Burp, and web rooms continue as practical structure.
- PortSwigger remains active every week to keep web testing sharp.
- Continued Linux shell repetition supports enumeration and first footholds.

## Phase 2 - Intermediate Exploitation
- HTB Academy becomes the main structured learning spine in Weeks 9-14.
- HTB or equivalent sanctioned easy targets provide more independent practice.
- PortSwigger continues for intermediate web issue repetition.
- Linux and Windows privesc modules plus linpeas and winpeas become weekly habits.

## Phase 3 - OSCP Track
- OffSec PEN-200 becomes the main structured exam-track resource in Weeks 15-24.
- Proving Grounds Practice and current TJ Null-style OSCP-aligned easy-to-lower-intermediate targets provide practical volume.
- OSCP-style report fragments, same-day writeups, and timed drills become part of the normal cadence.
- Weeks 20-23 emphasize reporting and mock work explicitly.
