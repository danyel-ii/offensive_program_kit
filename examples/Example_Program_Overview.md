# Program Overview Example

## Goal
Move from structured foundations to OSCP-track offensive readiness in 24 weeks.

## Phases
1. Foundations (1–4)
2. Offensive Entry (5–8)
3. Intermediate Exploitation (9–14)
4. OSCP Track (15–24)

## Weekly rhythm
- Mon/Tue/Thu: short controlled blocks
- Wed/Sat/Sun: deeper practical work
- Fri: rest

## Outputs
- learning logs
- weekly reviews
- machine writeups
- exploit attempt notes
- report fragments
